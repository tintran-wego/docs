## Team

The Flight Shopping team is responsible for Wego's flight metasearch functionality. We build and maintain the systems that:

- **Search & aggregate** flight fares from 400+ providers worldwide
- **Power Google Flights** integration (GFS) - a key revenue driver
- **Optimize user experience** with fast search results and accurate pricing
- **Maintain fare caches** for instant pricing and availability

**Team Lead:** Tin

## Key Systems

| System | Purpose | Tech Stack |
|--------|---------|------------|
| [Curiosity](/architecture/services#curiosity) | Main search orchestrator | Java, Dropwizard, Redis |
| [Flight Integrations](/architecture/services#flight-integrations) | Provider API integrations | Java, Dropwizard |
| [Flight Offer Engine](/architecture/services#flight-offer-engine) | Offer processing & caching | Spring Boot, ScyllaDB |
| [GFS](/architecture/services#gfs) | Google Flights integration | Java, Dropwizard |
